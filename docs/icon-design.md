# Knowsun Icon Design Session

## Summary

Designed a website favicon and icon set for **Knowsun Consulting**, an AI/ML boutique consulting firm based in Ann Arbor, Michigan.

---

## Brand Context

- **Name origin:** Derived from the Chinese character 智 — upper half means "knowledge," lower half depicts a "sun." Conveys AI/ML and intelligence.
- **Color palette:** Dark navy `#0C1B3A`, electric blue `#1E8FFF` / `#00C8FF`, white text.
- **Homepage tone:** Confident, direct, AI-first, human-centered.

---

## Icons Created

### Version 1 — Split Layout (智 structure)
- Upper half: stylized **K** with glowing neural node dots → represents *knowledge*
- Lower half: radiant **sun burst** in electric blue → represents *sun*
- Horizontal divider stroke mirrors the separating line in 智
- Files: `knowsun-icon.svg`, `knowsun-favicon-32.svg`

### Version 2 — Sun Circle + K (alternate)
- **Sun ring** with 8 radiating rays forms the outer circle
- Bold **K** centered inside the circle
- Glowing dots at K arm tips echo a neural/intelligence motif
- Files: `knowsun-icon-v2.svg`, `knowsun-favicon-v2.svg`
